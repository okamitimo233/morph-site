# Design Context

## Users

**目标用户**：开发者（项目创作者本人）

**使用场景**：设计风格测试探索，主观审美验证

**核心目标**：探索一套现代化、有辨识度的 UI-UX 设计系统，以个人博客为载体进行实验和迭代

---

## Brand Personality

**3 个关键词**：现代化 · 精致 · 个性化

**情感目标**：

- 让人感受到用心打磨的细节
- 既有秩序感又不失温度
- 每个交互都有"小惊喜"

---

## Aesthetic Direction

### 基调：现代精致风

**核心特征**：

- 圆角卡片与按钮（soft but not bubbly）
- 高级模糊效果（backdrop-blur, glass morphism 适度使用）
- 微妙渐变与阴影层次
- 流畅的动画过渡

### 背景策略

| 方案          | 描述                                     |
| ------------- | ---------------------------------------- |
| **流动渐变**  | 代码渲染的动态渐变背景，缓慢流动、呼吸感 |
| **大图 Hero** | 特定页面使用高质量图片作为视觉焦点       |

### 滚动交互

- 视差效果（parallax）
- 元素渐入渐出
- 状态随滚动变化（如导航栏背景、按钮样式）

### 反风格（Avoid）

| 不要             | 原因           |
| ---------------- | -------------- |
| 千篇一律的模板感 | 缺乏个性       |
| 极简到空洞       | 失去精致感     |
| 过度平面化       | 缺乏层次与深度 |
| 冷冰冰的科技感   | 缺乏温度       |
| 霓虹光污染       | 俗套、廉价感   |

## Theme

**支持模式**：

- 浅色主题
- 深色主题
- 手动切换
- 跟随系统偏好

**切换方式**：提供主题切换器组件，记忆用户选择

---

## Design Principles

### 1. 细节即品质

每个微小的交互、过渡、阴影都要精心设计。用户可能说不清哪里好，但能感受到"这就很精致"。

### 2. 动态但克制

动画服务于体验，不是为了炫技。流动的背景、渐入的元素要有节奏感，不是无止境的动。

### 3. 温暖的现代感

现代不等于冷冰冰。用圆角、柔和的颜色过渡、舒适的间距营造温度感。

### 4. 个性化但不怪异

可以有独特的视觉语言，但要在可接受的审美范围内。大胆但不冒犯。

### 5. 层次分明

通过模糊、阴影、渐变创造视觉层次，不是所有元素都平等对待。

## Technical Notes

**可用技术**：

- Tailwind CSS 4（@theme 定义设计 tokens）
- GSAP（动画库，已安装）
- React 19（交互组件）
- Astro Islands（按需水合）

**动画建议**：

- 页面入场：staggered reveal
- 背景：GSAP + SVG 或 CSS 动画的流动渐变
- 滚动：Intersection Observer 触发状态变化
- 主题切换：平滑过渡动画

## Color Palette Guidelines

**使用 OKLCH** 色彩空间（感知均匀）

**浅色主题**：

- 背景带轻微色偏（暖灰或冷灰，根据品牌色）
- 文字对比度充足
- 强调色克制使用

**深色主题**：

- 不是纯黑，是深灰带色调
- 避免高饱和度颜色在深色背景上刺眼
- 模糊效果在深色模式下更突出

---

## Typography Guidelines

**不要使用**：

- Inter, Roboto, DM Sans, Outfit 等过度使用的字体

**探索方向**：

- 正文：清晰易读但有特色的 sans-serif
- 标题：有性格的 display font
- 考虑中文场景的字体选择

---

## Animation Guidelines

**推荐**：

- ease-out-quart / ease-out-expo 用于入场
- CSS transform 和 opacity（不动画 layout 属性）
- 尊重 prefers-reduced-motion

**避免**：

- bounce / elastic 弹性效果
- 无意义的持续动画
